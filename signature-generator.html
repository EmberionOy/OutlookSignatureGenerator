How to add this to Outlook Classic (Windows)
  1. Click Copy signature above.
  2. Open Outlook → File → Options → Mail → Signatures.
  3. Select your signature or create a new one.
  4. Click inside the signature edit area and press Ctrl+A to select all, then Ctrl+V to paste.
  5. Click Save, then OK.

Note: the preview here uses placeholder image filenames. Make sure your actual image files are present in the Signatures\YourName_files\ folder, or replace the src values with hosted URLs.
